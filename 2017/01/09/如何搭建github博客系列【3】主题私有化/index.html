前言

个人博客信息

首先将博客根目录下的_config.yml配置文件中需要自定义的部分弄好：

title: 飞花逐月
subtitle: 我的个人博客
description: 学习总结 思考感悟 知识管理 展示交流
author: AlbertLee
email: 1075166573@qq.com #联系邮箱
language: zh-CN
timezone:
deploy:
type: git
repository: git@github.com:A1bertLee/A1bertLee.github.io.git
branch: master

主题自定义配置

进入/themes/Tinnypp/_config.yml配置文件中：

menu:
首页: /
归档: /archives
关于我: /about ##不需要可以不用加
profile:
enable: true
href: https://a1bertlee.github.io/
rss: /atom.xml ## RSS address.
imglogo:
enable: false ## display image logo true/false.
src: img/logo.svg ## .svg and .png are recommended,please put image into the theme folder /Tinnypp/source/img.
favicon: img/favicon.ico ## size:32px32px,.ico is recommended,please put image into the theme folder /Tinnypp/source/img.
apple_icon: img/myLogo.jpg ## size:114px114px,please put image into the theme folder /Tinnypp/source/img.
author_img_enable: true ## display author avatar picture
dataURI: false
author_img_data: ‘’
author_img: img/myLogo.jpg ## size:220px*220px.
author:
google_plus: ## eg:116338260303228776998 for https://plus.google.com/u/0/116338260303228776998
intro_line1: ‘AlbertLee’ ## eg: “Hello ,I’m Larry Page in Google.”
intro_line2: ‘编程爱好者’ ## eg: “This is my blog,believe it or not.”
weibo: ## e.g. 436062867 for http://weibo.com/436062867
twitter: ## e.g. yangjiansky for https://twitter.com/yangjiansky
github: ‘A1bertLee’ ## e.g. A1bertLee for https://github.com/A1bertLee
facebook: ## e.g. yangjian for https://favebook.com/yangjian
tsina: ## e.g. 1664838973 Your weibo ID,It will be used in share button.
duoshuo:
enable: true ## duoshuo.com
short_name: a1bertlee ##your duoshuo short name.

自定义评论功能

随后在themes/Tinnypp/source/js/embed.js这个文件中第二个函数中的判断e.user_id部分加上自己的多说ID。
如果你需要做评论中突出自己的博主身份，则在themes\Tinnypp\layout_partial下的after_footer.ejs文件中将140行那里按照提示做更改即可。
